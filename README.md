# Hashcode 2018

---

Installer le transpileur **TypeScript**

```bash
npm install -g typescript
```

Ajouter le format OnSave

```json
"editor.formatOnSave": true
```

Pour le dev, lancer dans un premier terminal :

```bash
tsc -w
```

Puis

```bash
node app.js
```
